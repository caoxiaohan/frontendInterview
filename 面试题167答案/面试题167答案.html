一、什么是bootstrap,bootstrap的特点？
    1 什么是bootstrap
	     Bootstrap是Twitter推出的一个用于前端开发的开源工具包,是基于 HTML、CSS、JavaScript 的前端框架。
	  2 bootstrap的特点
	    2.1 优点
          2.1.1 灵活简洁，代码优雅，美观大方
          2.1.2 栅格系统：响应式设计
                可以根据用户屏幕尺寸调整页面，方便做响应式布局。
          2.1.3 CSS模块化：提供一套优雅的 HTML+CSS 编码规范
                Bootstrap预先定义了很多CSS类，使用的时候直接给class赋予对应的类名。
                例如：button按钮，直接添加btn类名即可。
          2.1.4 javascript插件:让我们的 Web 开发更简单，更快捷
                不用自己写js，只需参照官方文档实例即可，使web开发更简单。
                例如：模态框，只要按要求设置好class，data-toggle，data-target即可。
          2.1.5 生态圈火，不断地更新迭代
      2.2 缺点
          2.2.1 存在浏览器兼容问题
                2.2.1.1 盒模型问题
                        问题：Bootstrap将所有的元素盒模型都设置成了border-box，这是IE混杂模式下的盒模型，光这点就导致了不能兼容IE。
                        解决：让IE浏览器按最新版本渲染
                              
                2.2.1.2 Bootstrap使用了html5新标签以及css3新语法
                        问题：IE8及其以下版本IE浏览器不支持html5新标签以及css3新语法
                        解决：引入html5shiv.js文件，
                2.2.1.3 Bootstrap判断用户屏幕尺寸用到了媒体查询
                        问题：IE8及其以下版本IE浏览器不支持媒体查询功能
                        解决：引入respond.js文件，
          2.2.2 依赖jQuery
                Bootstrap框架中的所有JS组件都依赖于jQuery实现。
                使用Bootstrap前必须引入jQuery文件。
                引入其他文件，势必导致加载速度变慢，影响用户体验。

七、常用web字体格式及不同浏览器兼容情况?
    1 常用web字体
      1.1、TureTpe(.ttf)格式：
        .ttf字体是Windows和Mac的最常见的字体，是一种RAW格式，因此他不为网站优化,支持这种字体的浏览器有【IE9+,Firefox3.5+,Chrome4+,Safari3+,Opera10+,iOS Mobile Safari4.2+】；
      1.2、OpenType(.otf)格式：
        .otf字体被认为是一种原始的字体格式，其内置在TureType的基础上，所以也提供了更多的功能,支持这种字体的浏览器有【Firefox3.5+,Chrome4.0+,Safari3.1+,Opera10.0+,iOS Mobile Safari4.2+】；
      1.3、Web Open Font Format(.woff)格式：
        .woff字体是Web字体中最佳格式，他是一个开放的TrueType/OpenType的压缩版本，同时也支持元数据包的分离,支持这种字体的浏览器有【IE9+,Firefox3.5+,Chrome6+,Safari3.6+,Opera11.1+】；
      1.4、Embedded Open Type(.eot)格式：
        .eot字体是IE专用字体，可以从TrueType创建此格式字体,支持这种字体的浏览器有【IE4+】；
      1.5、SVG(.svg)格式：
        .svg字体是基于SVG字体渲染的一种格式,支持这种字体的浏览器有【Chrome4+,Safari3.1+,Opera10.0+,iOS Mobile Safari3.2+】。
    2 浏览器兼容性
　　使用CSS3的@font-face属性可以实现在网页中嵌入任意字体，但是IE只支持微软自有的EOT格式字体，而其他浏览器都不支持这一字体格式，其它浏览器可以设置TTF(TrueType)和OTF(OpenType)两种字体作为自定义字体，浏览器对@font-face属性及各种字体格式支持详的细情况如下：
    Browser @font-face  TrueType  WOFF   EOT SVG  SVGZ
     IE        4+          9+      9+     4+     
     Firefox   3.5+        3.5+    3.6+       
     Chrome    4+           4+      6+        4+    6+
     safari    3.1+        3.1+     6+        3.1+  3.1+
     opear     10+         10+     11.1+      10+   10+
